два репозитория - домашний и удаленный

1. копия ссылки с гитхаба на репозиторий
2. домашняя папка
3. cd .. - выйти из текущей папки
4. cd имя_папки - зайти в папку
5. git clone путь репозитория  и название папки куда клонировать
6. после изменений  в папке клонированного - как обычно коммитить
7. git push - сохраненить его на гитхабе
8. зайти в домашнюю папку
9. git pull - слить измения с домашней папкой
